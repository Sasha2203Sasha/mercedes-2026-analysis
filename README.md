# Mercedes-AMG F1 2026: Mid-Season Strategy Analysis

## О проекте
Анализ гоночного темпа и деградации шин пилотов Mercedes-AMG в сезоне-2026.
Первые 8 этапов. Сравнение Джорджа Рассела и Кими Антонелли.

## Данные
- **Источник:** FastF1 API (официальные данные Formula 1)
- **Период:** Австралия – Австрия 2026
- **Количество кругов:** ~800 после очистки

## Методология
1. Загрузка сырых данных через FastF1
2. Очистка: удаление пит-стопов, кругов под сейфти-каром, выбросов
3. Расчёт коэффициента деградации шин (Linear Regression)
4. Визуализация: бокс-плоты темпа, бар-чарты деградации

## Ключевые выводы
- **Рассел** стабильнее в гоночном темпе (меньше разброс времени круга)
- **Антонелли** показывает более узкий диапазон времен, но чуть выше медиана
- Деградация шин: разница ~0.0X сек/круг (см. итоговый вывод в проекте)
