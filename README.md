# Mercedes-AMG F1 & Scuderia Ferrari HP 2026: Mid-Season Strategy Analysis

## О проекте
Анализ гоночного темпа и деградации шин пилотов Mercedes-AMG и Scuderia Ferrari HP в сезоне-2026.
Первые 8 этапов. Сравнение Джорджа Рассела и Кими Антонелли, а также Шалря Леклерка и Льюиса Хэмильтона.

## Данные
- **Источник:** FastF1 API (официальные данные Formula 1)
- **Период:** Австралия – Австрия 2026
- **Количество кругов:** ~1900 после очистки

## Методология
1. Загрузка сырых данных через FastF1
2. Очистка: удаление пит-стопов, кругов под сейфти-каром, выбросов
3. Расчёт коэффициента деградации шин (Linear Regression)
4. Визуализация: бокс-плоты темпа, бар-чарты деградации

## Ключевые выводы
- это вы можете узнать в проекте
- Деградация шин: разница ~0.0X сек/круг (см. итоговый вывод в проекте)
- КИМИ ЛУЧШЕ ВСЕХ(простите)
